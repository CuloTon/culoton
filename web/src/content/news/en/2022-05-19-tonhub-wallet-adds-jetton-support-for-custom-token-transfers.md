Tonhub, a widely-used TON blockchain wallet, has rolled out support for Jettons—the standard for custom tokens on the TON network. The update represents a milestone for user-facing adoption, as Tonhub becomes the first mass-market wallet to offer streamlined Jetton transfers without requiring technical expertise.

Jettons function similarly to ERC-20 tokens on Ethereum or BEP-20 on Binance Smart Chain, allowing developers to issue custom assets on TON. Until now, managing these tokens across wallets has been fragmented, with support scattered across specialist or developer-focused applications.

Tonhub's integration lowers the barrier for retail users to hold, send, and receive Jetton-based tokens. The wallet's consumer-friendly interface means that users navigating the TON ecosystem no longer need to juggle multiple applications for different asset types. This capability is particularly relevant as the TON ecosystem expands beyond native Toncoin into application tokens, staking derivatives, and community currencies.

The timing reflects broader momentum in the TON developer community to standardize and professionalize token infrastructure. As more projects launch on TON—from decentralized exchanges to gaming platforms—demand for accessible Jetton management grows correspondingly.

For wallet users, the practical benefit is straightforward: a single interface for both native and custom token transactions. For the ecosystem, it signals that user experience is becoming a competitive focus, not an afterthought. The integration may also encourage more projects to issue Jettons rather than alternative token formats, strengthening standardization across TON's token layer.
